# Routes de l'application

| Path            | Name        | Composant      | Description                               |
|-----------------|-------------|----------------|-------------------------------------------|
| /               | home        | HomeView       | Page d'accueil                            |
| /museum/:id     | museum      | MuseumRoomsView | Vue des salles d'un musée                |
| /room/:id       | room        | RoomView       | Détail d'une salle                        |
| /memory/:id     | memory      | MemoryDetail   | Détail complet d'une mémoire              |
| /modal          | modal       | ModalView      | Fenêtre modale pour formulaire ou détail  |
